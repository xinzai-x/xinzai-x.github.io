变量、函数在不同.c文件中使用

extern本质：告诉编译器 变量、函数 来至其他源文件

注意：static用于全局变量或函数前时，它将该变量或函数的可见性限制在定义它的文件内部

如果.c文件中的变量或函数，在对应的.h文件中声明了变量或函数，在别的 .c 文件中只需要生命对应的头文件，即可访问别的.C文件中的函数或变量，若不生命对应的头文件，则需要时候外加extern 来声明变量或函数
