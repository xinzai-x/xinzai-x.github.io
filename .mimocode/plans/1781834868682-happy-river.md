# Ink 主题优化计划

## 需求
1. 电脑端表格字体减小一号 — ✅ 已完成
2. 代码块复制按钮从底部移到右上角 — ❌ 未生效
3. 代码块左上角显示代码语言类型标签 — ❌ 渲染有误

---

## 问题分析

### 问题1：复制按钮仍在底部

**已尝试的方案**：
- `highlight.css:50-69` 添加 `.code-copy-btn { position: absolute !important; top: 8px !important; right: 8px !important; }`
- `main.js:398` 设置 `block.style.position = 'relative'`
- 均未生效，按钮仍在代码块底部中央

**可能原因**：
- `style.css` 中有更高优先级的规则覆盖了 `highlight.css`
- 需要检查 `style.css` 中是否有 `.code-copy-btn` 或相关选择器

### 问题2：语言标签与复制按钮文字合并

**当前实现**：
- `main.js:436-443` 从 `figure.className` 提取语言（如 "plaintext"）
- 创建 `<span class="code-lang-label">` 并 `block.appendChild(label)`
- `highlight.css:86-99` 定义 `.code-lang-label { position: absolute; top: 8px; left: 12px; }`

**问题**：截图显示 "复制plaintext" 合在一起，说明：
- 标签 span 可能未正确创建为独立节点
- 或 `position: absolute` 未生效，标签在文档流中与按钮相邻

### Hexo ink 主题代码块 HTML 结构

```html
<figure class="highlight shell">
  <table>
    <tr>
      <td class="gutter">行号</td>
      <td class="code"><pre>代码</pre></td>
    </tr>
  </table>
</figure>
```

**注意**：ink 主题**没有 `<code>` 元素**，语言在 `<figure class="highlight shell">` 的 class 上。
