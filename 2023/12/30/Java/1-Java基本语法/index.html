构造方法

注意：

构造方法名称必须与类名一致

构造方法名称前不能有任何返回类型的声明

不能在构造方法中使用return返回一个值，但可以单独写return语句作为方法结束

Java中每个类至少有一个构造方法，如果在一个类中没有定义构造方法，系统会自动为这个类创建一个默认的构造方法

方法重载

每个方法的方法签名是唯一【方法签名 == 方法名+参数列表】

重载：方法名相同，参数列表不同（只要满足其一即可：“参数定义类型不同、参数定义个数不同、参数定义顺序不同”）

作用：减少同样逻辑方法名字的定义（就是方法参数不同，实现逻辑是一样的）编译器会自动选择最合适方法进行操作

使用重载，完成两个数的求和

方法递归

递归：在方法内部再次调用自身方法

注意：设置有效的出口条件，可以让调用链上的每个方法都可以正确返回，避免无穷递归

可变参数

参数的个数是动态的，可有可没有，可以多个

必须放在参数列表末尾（即一个方法只能有一个可变参数）

可以直接当做是一个数组来使用（可变参数不能等同于数组）
